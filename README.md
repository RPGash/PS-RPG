Project "PS-RPG", an RPG text adventure created in PowerShell.

Alpha build. ~~May~~ Does contain bugs. Currently only works on Windows. Not tested on MAC or Linux.

Requirements
- PowerShell 7+
- PowerShell Module PSWriteColor
